Lab Environment Setup & Deployment
Lab provisioning and setup include one or more of the following components:
• ARM template deployment
• Custom Script Extension (CSE)
• Custom image-based environment setup
• Supporting deployment configurations as required

Package Scope Notes
• ARM deployment provisions only the Azure LabVM and required VM-associated dependencies such as networking and access components.
• No Azure Cosmos DB account, database, container, Azure AI resources, or other semantic-search workload services are pre-deployed by ARM.
• Learners create the Azure Cosmos DB for NoSQL semantic search workload resources themselves during Microsoft Tasks 1 through 5.
• The lab guide preserves the Microsoft semantic search task structure and ordering unchanged, adding only CloudLabs environment-access and tooling notes.
• Inline validations map to LabVM readiness, Cosmos DB/vector setup progress, and final completion state for the Microsoft task flow.

Exclusions
This package does not include:
• Scoring or grading mechanisms for inline validations
• Complex or advanced inline question types
